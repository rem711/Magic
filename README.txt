Le jeu utilise de base l'IA difficile. Pour passer à l'IA facile il faut modifier dans le code.
  --> magic.main, RunMagic.java 
  --> retirer les commentaires de la ligne 24 et les ajouter à la ligne 25.

Le jeu se joue en deux clics. Un premier pour sélectionner la carte, et un second pour la faire se déplacer.
Par exemple pour invoquer une créature, sélectionner la créature puis cliquer sur le terrain.
Les créatures se positionnent sur le premier emplacement libre du terrain et de la zone de combat.
Les enchantement s'appliquent à la première créature rencontrée sur le terrain.
